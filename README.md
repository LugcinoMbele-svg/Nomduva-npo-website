## 🚀 Getting Started

### Option 1: View Online (Easiest)
1. Go to Settings → Pages in your repository
2. Enable GitHub Pages on the main branch
3. Your website will be live at: `https://lugcinombele-svg.github.io/Nomduva-npo-website/`

### Option 2: View Locally
1. Clone the repository
2. Open `index.html` in your web browser
3. No installation or server needed!

## 📝 How to Customize

### Update Organization Details
Edit the following in `index.html`:

**Contact Information:**
```html
<p><a href="mailto:info@nomduva.org">info@nomduva.org</a></p>
<p>+27 (0) XX XXX XXXX</p>
```

**Statistics:**
```html
<h4>1000+</h4> <!-- Change this number -->
<p>People Helped</p>
```

**Services:**
Add or modify service cards in the Services section.

### Customize Colors
Edit the color variables in `styles.css`:
```css
:root {
    --primary-color: #2c5f2d;      /* Main green */
    --secondary-color: #97bc62;    /* Light green */
    --accent-color: #ff6b6b;       /* Red accent */
}
```

### Add Your Images
Replace placeholder images:
```html
<img src="https://via.placeholder.com/300x200?text=Education" alt="Education Impact">
```

With your own:
```html
<img src="path/to/your/image.jpg" alt="Education Impact">
```

## 🔗 Setting Up Contact Form

To make the contact form send emails, you can use:

1. **Formspree** (Recommended for beginners)
   - Go to formspree.io
   - Create an account
   - Update the form action in `index.html`

2. **EmailJS**
   - No backend server needed
   - Easy setup with JavaScript

3. **Backend Solution**
   - Use services like Netlify, Vercel, or Firebase

## 🌐 Deploy Your Website

### Option 1: GitHub Pages (Free & Easy) ⭐ RECOMMENDED
1. Go to your repository Settings
2. Scroll to "Pages" section
3. Select "main" branch
4. Your site will be live in 1-2 minutes!
5. Visit: `https://lugcinombele-svg.github.io/Nomduva-npo-website/`

### Option 2: Netlify (Free)
1. Go to netlify.com
2. Click "New site from Git"
3. Connect your GitHub repository
4. Done! Your site is live

### Option 3: Vercel (Free)
1. Go to vercel.com
2. Import your GitHub repository
3. Deploy automatically
